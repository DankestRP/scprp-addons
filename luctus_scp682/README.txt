# Luctus SCP682

You need to change job to 682 for this scp to work correctly. Many features are bound to a job change.

The SWEP has the following functions:

 - Leftclick: Short range attack
 - Rightclick: Open Doors
 - Reload: Jump forward

The SWEP was tested with the following playermodel (which needs to be added to the server workshop collection):

https://steamcommunity.com/sharedfiles/filedetails/?id=905114757
