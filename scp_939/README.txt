# Luctus SCP939

You need to change job to 939 for this scp to work correctly. Many features are bound to a job change.

As SCP939 players are invisible except if they:

 - Shoot
 - Talk
 - Walk

The SWEP has the following functions:

 - Leftclick: Short range attack
 - Rightclick: Special ability, mark all nearby players (default radius: 2048)
 - Reload: Scream, which makes your vision brighter for a short time
